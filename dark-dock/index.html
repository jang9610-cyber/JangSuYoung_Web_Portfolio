🐕 경비견 (GuardHound)
지상 순찰 + 플레이어 감지 시 추적 → 돌진 공격. 소음에 반응하여 위치 이동.
🌿 파리지옥 (AmbushPlant)
천장 매복. 하단 통과 시 낙하 공격 + 기름 드롭 연출. 체력이 높아 회피 우선.
📹 밀고자/CCTV (Squealer)
감시 후 단계별 경고. 위험 단계에서 경비견 소환. 직접 공격 없음.
아이템효과주요 파라미터

🔫 조명탄발사 → 넓은 범위 밝히기발사력 10, 머즐 파티클
