## 📖 게임 스토리

이 공장은 더 이상 인간을 필요로 하지 않는다. 모든 생산, 관리, 보안은 완전 자동화 시스템에 의해 통제되고 있으며, 인간의 출입은 엄격히 금지되어 있다.

그러나 단 하나의 예외가 발생한다. 회사의 '1급 자산'이 외부로 반출되어야 하는 상황. 그 자산은 다름 아닌, **회장 개인 소유의 황금 골프채**였다.

회사는 이를 회수하기 위해 정규 직원도, 특수 요원도 아닌 **최저시급 알바생, 당신**을 투입한다! 그에게 주어진 임무는 단순하다.

> **"자산을 확보하고, 살아남고 탈출하라!"**

## ✨ 흥미요소

- 플레이어는 자동화된 공장 내부에 진입하며, **제한된 시야와 최소화된 정보**로 인해 지속적인 긴장감을 느끼게 된다.
- **2개의 아이템 슬롯**만을 활용해 생존한다는 조건과 외부로 탈출해야 한다는 명확한 목표는 강한 도전 욕구를 유발한다.
- 적의 위치와 상태를 알 수 없는 상황에서 어떻게 위기를 극복할 것인지에 대한 선택은 즉각적인 공포와 함께 **전략적인 판단의 긴장감**을 제공한다.
- 이러한 선택과 제약을 극복하고 탈출에 성공했을 때, 플레이어는 높은 난이도를 돌파했다는 **강한 성취감**을 느낄 수 있다.

## 🎨 게임의 컨셉 (기획의도)

| 항목 | 설명 |
| --- | --- |
| **장르** | 사이드 스크롤링 슈팅 액션 |
| **스토리** | 공장에 갇힌 플레이어는 각종 무기와 도구들로 목숨의 위협으로부터 자신을 지키며 밀실 공간을 탈출하기 위한 모험을 하게 됩니다. |
| **주인공** | 골프채를 가지고 공장을 탈출해야 한다는 역할을 수행하며, 궁극적으로 탈출을 목표합니다. |
| **최종 목표** | 공장의 출구를 열기 위한 열쇠를 얻어, 출구를 열고 나간다. |

### 게임의 특징

- 공포 게임의 요소를 오마주하여 플레이어의 행동과 정보를 제한함으로써 **모바일에서도 공포 게임의 분위기**를 느끼게 함.
- 제한된 플레이로 문제 상황을 해결했을 때 오는 **성취감**을 제공하고자 함.

### 기타 컨셉 요소

- **아이템 획득:** 특정 효과를 일으킬 수 있는 아이템을 소지하거나 사용할 수 있음
- **지형 상호작용:** 플레이어의 행동이 지형 요소와의 상호작용을 일으켜 위기 상황을 타파하거나 우회하게 만들 수 있음

## 🧩 게임의 요소

### 플레이어

- 총을 소유하고 있고, 총을 발사함으로써 피해를 줄 수 있음
- 아이템을 줍고 상호작용 사용 가능
- 인벤토리는 **2개의 슬롯**이 제공됨

### 적

- 적들은 게임 시작 시 맵에 배치되고, 각각 고유한 패턴이 존재함
- 플레이어가 총 혹은 아이템의 고유 기능으로 체력을 소진시킬 수 있으며, 체력 모두 소진 시 사망

### 아이템

| 아이템 | 설명 |
| --- | --- |
| **열쇠** | 사용 가능한 곳에서 자동으로 사용됨 |
| **투척형 아이템** | 플레이어가 사용 시 던질 수 있음. 각 투척 아이템은 고유의 기능을 보유 |
| **총알** | 인벤토리에 들어가지 않고 플레이어의 총알 소지 수에 가산됨 |
| **소지 효과 아이템** | 소지만으로 효과 발동 (예: 탄창 소지 시 총알 최대치 상승) |

## 🔧 핵심 기술

- Game Start / Game Over 화면 UI 구현
- Player 상태 (HP, 스태미나, 총알 개수) UI 구현
- Player와 Enemy 공격 및 피격 음향 구현
- Shooting 기반의 발사 효과 (음향, 파티클 등) 구현
- 게임 컨셉(공포)에 어울리는 에셋, 배경, BGM 사용
- 퀵슬롯 구현, 다양한 아이템 습득 및 사용 효과 구현
- W키를 누를 시 상단 시야 확장

## 📝 특이사항

- 게임의 분위기를 조성하기 위해 **그래픽적 요소를 적극적으로 활용**할 것임.
- 소지 아이템 수를 제한시킴으로 인해 **전략적인 선택 요소**를 추가하고, 구현 기능을 축소시켜 개발력을 집중시킴.
